A comprehensive suite of tools to customize every aspect of your SP Tarkov experience.

"Rebuild the trader economy from scratch, define custom player profiles, tier every item in the game by rarity and value, enrich item descriptions with crafting and barter info, fine-tune hideout requirements, override handbook prices, control insurance, manage bot loot — and more. All in one place, all independently configurable, all optional."

"Over the past few weeks I've been releasing a handful of separate mods — each built around the same philosophy, the same config system, and designed to work together. Maintaining them independently was becoming a pain, so I merged everything into a single coherent package. Same features, same logic, cleaner experience."

"RZEssentials is built for players who want to craft their own experience and aren't afraid to dig into config files. That said, it's designed to be approachable — every feature ships disabled by default, so installing it changes absolutely nothing until you decide otherwise.
Each config file maps to one isolated system. Touching the trader economy won't affect your hideout. Tweaking item tiers won't touch your profiles. There are no hidden dependencies, no cascading side effects — just clean, independent controls.
The config files themselves are written to be read. Well organized, clearly commented, no cryptic values — if you've never modded before, you'll figure it out." This is a side project, but it's not treated like one."

"As with any mod, a backup before making big changes is always a good habit — but nothing in RZEssentials should ever be irreversible. If a feature has any kind of permanent effect on your save, it will be clearly flagged in the documentation. Otherwise, uninstalling or reinstalling the mod brings you straight back to a clean slate, no strings attached."

"That said, I get it — not everyone wants to spend an evening tweaking config files. For those who don't, I'm planning to eventually release a ready-to-play mod built on top of all these systems: fully configured, balanced, and drop-in ready.
RZEssentials exists because I needed these tools to build the version of Tarkov I actually want to play with my friends. Once we've had enough time to test and balance everything properly, that experience will be released as a complete overhaul. Consider this the foundation."

Limitations Most of this mod was designed around my own setup — and I run my server with the flea market disabled. That means flea-related features are limited and unlikely to be a development priority going forward. Most systems have nothing to do with it either way, but if you're also running a flea-free server, you'll find that a lot of things work especially well out of the box — because that's exactly the environment this was built for."

On the flip side — most of what's in here is genuinely unique. No equivalent in SVM, no equivalent in Profile Editor, no equivalent anywhere else. If you've been looking for a specific feature and couldn't find it, there's a decent chance it's in here."
